设置

控制扫描时同时工作的线程数，用来加快扫描。机械硬盘建议设为 1，固态硬盘可适当提高

扫描完成后把快照压小再保存，用来减少占用空间。对比时会自动解压

扫描整个盘符时，优先读取系统文件表来加快速度。需要管理员权限，且仅适用于 NTFS；失败时会自动改用常规扫描

开启后打开搜索会先载入索引，占用更多内存，但后续搜索更快。内存充足建议开启；内存紧张或小目录可关闭

快照存放目录

扫描得到的快照会保存在这里。更改后点完成，会把原目录里的快照迁到新位置

删除白名单

名单中的路径不会被删除

名单中的路径禁止删除，可选三种匹配： 精确：只保护这一条路径，例如 C:\Games\save.dat 前缀：保护该路径及其子项，例如 C:\Windows 会连同其下所有文件 正则：按正则匹配完整路径，例如 (?i)\\Windows\\ 可匹配路径中含 \Windows\ 的项

检查更新

从 GitHub 查询最新发布版本，并与当前版本比较
