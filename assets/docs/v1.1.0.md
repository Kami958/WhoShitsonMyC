# v1.1.0

### 新增

1. 新增对比树 **搜索功能**，允许用户输入关键字查询
2. 搜索支持区分大小写与严格整名匹配
3. 打开搜索时可选用内存索引预热，通用设置可开关（默认开启）
4. 设置页支持从 GitHub 检查更新，并可打开发布页
5. 对比树支持加入待删除列表；右侧工具侧栏批量删除（默认回收站，可选本会话彻底删除）
6. 设置页可维护删除黑名单（精确 / 前缀 / 正则），命中路径禁止删除
7. 可选 AI 模块：设置配置、侧栏对话、右键问 AI、流式回复；打包默认主线无 AI，另提供含 AI 包（`--with-ai`）

### 修复

1. 对比选择框有无备注时高度不一致，无备注时显示「无备注」

### 优化

1. 对比选择条改为基准/当前与日期同行，选中后边框着色，并与左侧新建扫描区同高
2. 空状态引导文案
3. UI优化，删除冗余功能
4. 前端逻辑由单体 app.js 拆到 web/js 多文件
5. 应用日志改为 stdlib logging 统一出口：等级门槛、DEBUG 样式、扫描计时并入 applog；开发用 `WSMC_LOG_LEVEL=DEBUG` 即可（`WSMC_SCAN_TIMING` 仅兼容）
6. 版本比较区分可升级、已最新、本机高于发布版；当前版本显示在检查更新旁，窗口标题不再附带版本
7. 展开工具侧栏时加宽应用窗口，避免挤压主内容；设置页默认尺寸加大
8. 设置变更统一走 `applog.log_settings_changed`（默认 DEBUG，只记改动字段；路径是否脱敏跟全局开关）
